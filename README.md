# Discord Embed IP Logger

Creates a link that appears broken in Discord but captures visitor IP information.

## Setup

1. Create a Discord webhook in your server
2. Replace `YOUR_WEBHOOK_URL` in `index.html` with your webhook URL
3. Host the files on GitHub Pages or any web host
4. Share the link in Discord

## Files

- `index.html` - Main page that captures IPs and fake breaks

## Usage

The link will show no preview in Discord. When clicked, it displays visitor IP and sends data to webhook.

## Note

For educational/testing purposes only. Not made for malicious purposes this is to show how they do work.
